# UPI-Transaction-Data-Analysis-
This project is an interactive UPI Transaction Analysis Dashboard developed using Microsoft Power BI. It analyzes UPI transaction data to identify monthly trends, transaction amounts, balance patterns, and customer behavior across different banks, cities, age groups, and payment methods. The dashboard uses Power Query for data cleaning and DAX for KPI calculations, allowing users to explore insights dynamically through interactive filters and visualizations.
